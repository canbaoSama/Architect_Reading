### 🎯 逻辑守则

*选股与节奏的底层逻辑 —— 热点板块 · 趋势方向 · 抄底节奏*

|  | 规则 | 说明 |
|:---:|:---|:---|
| ① | 只炒当下火热板块的巨头股票 | 拒绝冷板凳，半年都不动的票不碰 |
| ② | 只做上升趋势，适当做 T | 做 T 铁律：**一定要接回**，不接回等于踏空 |
| ③ | 大幅下跌可抄底 | 必须快进快出，抢反弹要果断，不恋战 |
| ④ | 波段内低点接入 | 走势太强时，小仓位试探，先看情况再加 |

---

### 📊 趋势守则

*根据走势形态决定进出场 —— 回落强度 · A/V 顶底形态*

|  | 场景 | 操作 |
|:---:|:---|:---|
| ① | 回落不明显，强势往上 | ✅ 可接入 |
| ② | 回落不明显，缓慢向下或平稳波动 | ⚠️ 先撤离 |
| ③ | 回落明显 | ⏸️ 不第一时间接，等企稳再进 |
| ④ | 大幅回落后，长时间拉起幅度较小 | ⚠️ 先撤离 |
| ⑤ | A 形为顶，双 A 也为顶；V 形为底，双 V 也为底 | 后一波压不过前一波 → ⚠️ 先撤离 |

### 📋 附录：可选用守则

*以下为常见风控与心理守则，可根据自身风格决定是否采纳。*

| 守则 | 内容 |
|:---|:---|
| 🛑 止损纪律 | 单笔亏损超过 X% 即止损，不幻想回本 |
| 📦 仓位管理 | 单只个股不超过总仓位 X%，分散风险 |
| 🧘 情绪隔离 | 不因一次大赚加仓、不因一次大亏报复性交易 |
| 🕐 时间窗口 | 避免在财报前、重大事件前重仓博弈 |
| 😴 空仓也是策略 | 没有好机会时，空仓等待比强行交易更明智 |
