# 项目亮点与技术选型

## 技术选型总览

| 场景 | 技术方案 |
|------|----------|
| 甘特图 | DHTMLXGantt、FrappeGantt |
| 看板 | ECharts、Grafana |
| 流程图 | G6、X6 |
| 2D 图形渲染 | Fabric.js、PixiJS、Leaferjs |
| 3D 地图仿真 | Leaflet、Cesium、Three.js、Mapbox GL JS、OpenLayers、ArcGIS |
| 路径规划 | GraphHopper |

## 项目一：图片编辑器

- 技术选型：比对 Fabric.js、PixiJS，选择 Leaferjs——轻量、性能好、学习成本低、支持跨平台
- 亮点：操作效率优先，常用操作比竞品更多；选框内固定提示组件的放大、缩小、旋转计算需额外开发
- 架构决策：提示组件与选框放同一 box，便于统一处理鼠标交互与选中判断

## 项目二：智能助手

- 集成 MCP 远程工具：model、memory、文档、prompt、agent 等
- 联调华为部署模型（qwen2.5 8b，mindie 部署）
- 预分类流程：先检索历史 → 再检索文档向量库 → 再决定是否调用远程工具
- 折叠面板展示多类型数据，可扩展接入 v0 等强模型做界面生成

## 项目三：Cesium 3D 仿真

- 态势绘制：点、线、面、圆、多边形、箭头（直线/攻击/钳击）、侦察矩形/锥形、导弹抛物线、雷达扫描
- 特殊材质：水波纹、流动线条、发光线条（GLSL）
- 场景特效：阳光、雨、雪、雾、全球水淹
- 仿真播放器：支持前进、后退、倍速播放

## 项目四：一站式开发管理平台

- 对接 GitLab、公司内部管理平台
- 看板界面（Grafana）
- 表格等多类型数据解析

## 技术难点攻克

### 1. Cesium 材质与特效
- 学习 GLSL 与 Shader Graph 思路，配合 AI 仿照开发
- 结合 Turf.js 做地理计算，自行实现特效

### 2. 图形平滑过渡
- 直接更新 → 闪烁；CallbackProperty → 跳动；requestAnimationFrame 切片 → 仍有跳动
- 最终方案：`Cartesian3.lerp()` + 格林时间间隔插值，平滑过渡

### 3. AI 智能体接入逻辑
- 预分类：历史、文档优先；仅在必要时调用工具
- 参考：阿里百炼、百度千帆、腾讯 TI、华为 ModelArts
