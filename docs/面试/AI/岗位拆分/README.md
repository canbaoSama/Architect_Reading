## 使用建议

- 先选 1 个主攻岗位，再选 1 个相邻岗位补充。  
- 每个岗位至少准备：2 个项目故事 + 1 张系统图 + 1 组量化结果。  
- 复习顺序建议：知识点清单 -> 基础题 -> 进阶题 -> 场景题。
