## 面试趋势（2025）

- **GenAI 占比约 75%**：RAG、提示词工程、Agent 设计成为主流考点，传统 ML（CNN、梯度下降等）约占 20-30%
- **前端转 AI 优势**：懂交互 + 懂 AI 的「AI 界面 specialist」需求旺盛
- **典型面试轮次**：4-6 轮，2-4 周，包含系统设计、深挖技术、行为面试
- **必考场景**：设计 RAG 客服聊天机器人、文档问答、多 Agent 协作系统

## 前端转 AI 核心能力要求

| 能力 | 说明 |
|------|------|
| 大模型与提示词 | 了解基本原理，能设计与优化 prompt |
| RAG | 能基于 RAG 搭建私有知识库问答 |
| Agent | 理解工作流程，会使用 FunctionCall |
| **主流框架** | 了解 OpenClaw、LangGraph、Rules/Skills、MCP 等，至少熟悉其一 |
| 前端+AI 项目 | 对话界面、流式输出、实时响应 |
| 方案设计与落地 | 需求拆解、AI 方案设计、上线迭代 |

## 常见岗位方向

- **AI 应用工程师**：调用 LLM API、搭建 RAG、集成到业务
- **前端 AI 工程师**：AI 交互、流式 UI、对话体验
- **AI 产品 / 全栈**：从产品到前端的完整 AI 体验
