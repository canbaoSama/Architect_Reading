## 面试趋势（2025）

- 从「背诵八股文」转向**工程能力与架构设计**
- 场景化系统设计题成为必考
- AI 工具改变考核标准：重点转向逻辑抽象与架构决策
- 关注底层原理而非 API 记忆

## 高频考点速览

- **JS**：事件循环、原型链、闭包、this、异步
- **Vue**：响应式、Composition API、性能优化
- **浏览器**：缓存、HTTP、安全
- **工程化**：性能优化、微前端、构建工具
